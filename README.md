# nekos.py
A Python module that uses Nekos API

# Install
```
pip install -U https://github.com/Nekos-life/nekos.py/archive/master.zip
```

# Usage example
```py
import nekos
print(nekos.cat())

>>> 'https://cdn.nekos.life/meow/04D5A.jpg'
```

NOTE: You must call them as functions
